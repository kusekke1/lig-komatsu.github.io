関数

目的別にいっぱあい用意されている

例えばCSVファイルを読み込む

read.csv("ファイル名")

これでcsvファイルにあるデータをRで加工することが可能になったりします

データをいじってみる

まずはCSVファイルの読み込み

read.csv("file_name")

これでCSVファイルを読み込みます。

x = read.csv("./classData.csv")

マルチバイト問題

日本語を使ったCSVファイルだと正しく読み込めないことがあります。

その場合、第二引数にfileEncoding = "cp932"を指定すればおｋれす

x = read.csv("./classData.csv", fileEncoding = "co932")

列を指定する

オブジェクト名$列名

平均値を出す

mean(x$Mathematics)

中央値を出す

median(x$Mathematics)
