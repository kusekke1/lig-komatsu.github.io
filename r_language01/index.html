みなさん

どれくらいショートカット使ってますか？？？

ぼくはショートカットが大好きマンです

なので

よく使うショートカットを紹介して、終わろうと思います。

Gmail

既知のことかもですがGmailのおすすめショートカット

上下移動

上：k

下：j

そう…Vimと一緒

全員へ返信

a

受信ボックス一覧

g → i

検索窓にフォーカス

/

全選択

* → a

全選択解除

* → n

削除

#

応用：全メール削除

* → a → #

ちなみに

Googleの検索結果画面でも同じようなショートカットが使えます。

ぼくがよく使うのは

検索窓にフォーカス： /

上下移動： j, k

ページを開く： Enter
